# Static and Default Routing Overview 

- Tižm hižžu kiž vžž static Routing. 
- Kižžm soažt lžu lžžžng važ thižžt lžžp default route ra ngoaži doanh nghižžp. 

## Static and Default Routing Overview
- Dužng trong tržžžng hžžp
    - Mž hižnh mažng nhož važ chiž yžu cžžu žžn giažn vžž žižnh tuyžžn 
    - Mž hižnh mažng Hub-and-Spoke (kžžt nžži tžžp trung)
    - Tažo nhanh `ad hoc route`
    - žžžžc dužng žžž lažm phžžng ažn džž phožng 
- Khžng nžn dužng trong tržžžng hžžp 
    - Mž hižnh mažng lžžn 
    - Mž hižnh mažng chža cžž žižnh, což thžž mžž ržžng. 
- Static route (žižnh tuyžžn tižnh) žžžžc sžž dužng trong tržžžng hžžp žižnh tuyžžn mažng lžžn žžžn stub network (truy cžžp bžži 1 žžžžng link duy nhžžt)
